11
Global rules
Apply to every screen, no exceptions.
i

No "X min read" anywhere

Strip all read-time meta. Bylines collapse to just the author name (e.g. "Brad Brown"), or the meta row is removed entirely.

ii

No photo captions anywhere

Italic caption lines below hero images and feature article images are removed. This includes Shu Shu article, Daiginjo piece, brewery heroes, event hero galleries.

iii

Phone frame: 42px radius + 1px inset rim

.phone{border-radius:42px; box-shadow:inset 0 0 0 1px #0a0a0a}. Eliminates rounded-corner edge artifact. Do not use 44px or 46px.

iv

"Community" never "Counter" as visible label

The file is still named sakeyo_Counter_TypeC.html (invisible to users), but every on-screen mention must say Community. Alerts forum card overline is "Community · New reply".

v

Sentence case everywhere

Only the SAKEYŌ wordmark is uppercase. Section titles, bottle names, article titles — all sentence case.

vi

代 footer on every screen

72px kanji in rgba(16,24,32,0.10) + "One rung at a time." in var(--ink-mute), centered.
